## 設計根拠

### 搭乗クラス（#1〜#2）
2種類の搭乗クラスをそれぞれ1件ずつ設定。同値分割法のみ（境界値なし）。

### ピーク時期（#3〜#5）
3区分（ピーク1・ピーク2・通常）を代表値1件ずつで同値分割法を適用。

### ピーク時期の境界値（#6〜#25）
ピーク1が4期間・ピーク2が4期間あるため、境界は合計10カ所。  
各境界につき「終了日」と「翌日」の2件を設定し、計20件。

| 境界 | 終了側 | 翌日側 | 種別 |
|---|---|---|---|
| 1/5 → 1/6 | ピーク1（140%） | 通常（100%） | ピーク1 → 通常 |
| 3/12 → 3/13 | 通常（100%） | ピーク2（125%） | 通常 → ピーク2 |
| 3/19 → 3/20 | ピーク2（125%） | ピーク1（140%） | **隣接：ピーク2 → ピーク1** |
| 3/31 → 4/1 | ピーク1（140%） | 通常（100%） | ピーク1 → 通常 |
| 7/17 → 7/18 | 通常（100%） | ピーク2（125%） | 通常 → ピーク2 |
| 8/7 → 8/8 | ピーク2（125%） | ピーク1（140%） | **隣接：ピーク2 → ピーク1** |
| 8/18 → 8/19 | ピーク1（140%） | ピーク2（125%） | **隣接：ピーク1 → ピーク2** |
| 8/31 → 9/1 | ピーク2（125%） | 通常（100%） | ピーク2 → 通常 |
| 12/18 → 12/19 | 通常（100%） | ピーク2（125%） | 通常 → ピーク2 |
| 12/25 → 12/26 | ピーク2（125%） | ピーク1（140%） | **隣接：ピーク2 → ピーク1** |

> **注：** 12/31 → 1/1 はどちらもピーク1（140%）のため境界なし。年をまたいでも区分は変わらない。

### 運賃の端数（#26〜#28）
「切り上げが発生しない（端数0円）」と「発生する（端数1円・99円）」の3点で境界値分析。  
端数の最小値（1円）と最大値（99円）を両端として設定した。
