# 課題9 不具合レポート

## BUG-001 子供料金が大人料金と同額で計算される

| 項目 | 内容 |
|------|------|
| 不具合ID | BUG-001 |
| タイトル | 子供料金（12歳未満）に割引が適用されず、大人料金と同額で合計金額が算出される |
| 重大度（Severity） | **A**（金銭への直接的影響） |
| ステータス | オープン |
| 発見日 | 2026-05-14 |
| 発見方法 | Playwright 自動テスト（TC-R-001） |

---

## 重大度の判定根拠

子供料金の割引が適用されないため、実際の正規料金より高い金額がユーザーに提示・請求される。
金銭的な過請求に直接つながる欠陥であり、**Severity A**（金銭への直接的影響）と判定する。

---

## 再現条件

- ログイン済み会員として予約操作を行う
- 搭乗者に12歳未満（子供）が1名以上含まれる
- 片道フライトを予約する

---

## 再現手順

1. `http://localhost:8083/atrs/` にアクセスする
2. 会員番号 `0000000001` / パスワード `aaaaa11111` でログインする
3. 空席照会画面（B101）で以下の条件を入力し「照会」ボタンを押す
   - 出発地：東京(羽田)
   - 到着地：大阪(伊丹)
   - フライト種別：片道
   - 搭乗クラス：一般席
   - 搭乗日：本日から7日後
4. 検索結果から最初のフライトを選択し「選択フライトを予約」ボタンを押す
5. お客様情報入力画面（B202）で「搭乗者を追加」ボタンを押す
6. 搭乗者2に以下を入力する
   - 氏名（カタカナ）：テスト コドモ
   - 年齢：**5**
   - 性別：男性
7. 「予約確認」ボタンを押す
8. 申し込み内容確認画面（B203）で合計金額を確認する

---

## 発生事象（Actual）

申し込み内容確認画面（B203）に表示された合計金額が **¥41,200**。

これは大人運賃（¥20,600）× 2 と等しく、子供（5歳）に対しても大人と同額の運賃が適用されている。


---

## 期待結果（Expected）

仕様：合計金額 = 運賃 × 搭乗者数（12歳以上）+ （基本運賃 × 60% − 割引額）× 搭乗者数（12歳未満）

12歳未満の搭乗者には「基本運賃 × 60% − 運賃種別ごとの割引額」が適用されるため、
合計金額は **大人運賃 × 2（¥41,200）より少ない金額** になるべきである。
