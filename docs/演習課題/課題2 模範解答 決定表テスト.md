## 設計根拠

### ルール数について

条件が5つあるため理論上の組み合わせは 2⁵ = 32 通りですが、以下の理由で6ルールに絞っています。

**無関係条件（`-`）を適用した理由：**

| 条件 | `-` にした理由 |
|---|---|
| ルール1の C2〜C5 | 代表者が18歳未満の時点で予約不可が確定するため、他の条件は結果に影響しない |
| ルール2・5・6の C3 | レディース割でない場合、性別条件は存在しないため無関係 |
| ルール2〜4の C5 | グループ割でない場合、人数条件は存在しないため無関係 |
| ルール3・4の C5 | レディース割とグループ割は同時選択不可のため無関係 |

**条件の依存関係：**

```
C1（年齢） が N → 即✗（C2〜C5 は関係なし）
C1（年齢） が Y
　├─ C2（レディース割）が Y → C3（性別）で判定
　├─ C4（グループ割）が Y → C5（人数）で判定
　└─ C2・C4 ともに N（通常運賃）→ 即✓
```

### 前提条件

- レディース割とグループ割は同時選択できない（運賃一覧 表3 より）
- グループ割の利用可能最少人数は3名（運賃一覧 表3 より）
- 予約代表者の年齢条件（18歳以上）はすべての運賃種別に共通して適用される
